# league-purple-theme

# Required
 * <a href="https://github.com/nomi-san/league-loader">League loader</a>

# How to install:
 * Copy purple-theme.css 
 * Copy `plugins` folder into league-loader directory
 * Copy `assets` folder into league-loader directory
